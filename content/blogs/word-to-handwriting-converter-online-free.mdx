## Common Mistakes to Avoid

### Pasting Too Much Text at Once

Long document sections can make handwriting pages look crowded. Split long content into smaller sections before converting.

### Choosing Decorative Fonts

Decorative cursive can look attractive, but it may be harder to read. Use simple handwriting styles for document text.

### Ignoring Line Spacing

If the lines are too close, the output feels artificial. Slightly wider spacing usually looks better.

### Downloading Without Previewing

Preview first. A quick check helps you catch layout issues before exporting the final file.

### Using the Tool Where Original Handwriting Is Required

Use handwritten-style output only where it is allowed. If original handwriting is required, follow that instruction and use the converter for planning, drafts, examples, or study notes instead. You can read the full [Responsible Use](/responsible-use) page for clear guidance.
